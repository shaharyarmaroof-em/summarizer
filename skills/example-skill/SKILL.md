# Skill: example-skill

## When to use
- You need a reference for how to structure a local skill.

## Workflow
1. Read this file to understand the required sections.
2. Copy `skills/_template` into a new folder under `skills/`.
3. Update `skills/registry.json` with the new skill entry.

## Inputs
- A short name for the skill.
- The problem the skill addresses.

## Outputs
- A new skill folder with a completed `SKILL.md`.
